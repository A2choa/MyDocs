    Pythonの概要¶

    Pythonとは¶

    • ブラウザ操作やデータ収集に特化した言語。日々の作業を自動化（Webスクレイピング）しよう

    動作環境¶

    • 以下のライブラリを主に使用する
      • Requests
        • HTMLファイルからデータを取得する
      • WebDriver
        • ブラウザを操作するAPIを公開する
      • selenium
        • WebDriverと連携しブラウザを操作する
      • beautiful soup
        • Requestsで取得したデータから必要なデータのみを取得する
